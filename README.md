# Pieczone z Sercem

Projekt strony internetowej dla piekarni `Pieczone z Sercem`.

Na tym etapie repozytorium zawiera fundament projektu:

- brief marki
- plan realizacji
- checklistę materiałów wejściowych

Dokumenty robocze:

- `docs/master-brief.md`
- `docs/project-plan.md`
- `docs/etap-1-foundation.md`

Założenia projektu:

- strona w języku polskim
- poprawne kodowanie UTF-8
- priorytet: mobile first
- nacisk na design, atmosferę i scroll-driven storytelling
- osobny moduł `Co dziś pieczemy`
