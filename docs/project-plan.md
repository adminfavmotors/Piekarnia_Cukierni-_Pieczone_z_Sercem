# Plan Realizacji

## Etap 1. Fundament

- zatwierdzić brief projektu
- potwierdzić finalną nazwę marki
- zebrać dane kontaktowe i godziny otwarcia
- zebrać wszystkie dostępne zdjęcia
- spisać historię piekarni i właścicielki
- przygotować listę wypieków i kategorii

## Etap 2. Content

- napisać finalne teksty po polsku
- przygotować szkic treści dla wszystkich sekcji
- przygotować przykładowe pozycje do `Co dziś pieczemy`
- ustalić etykiety sezonowe i opisy kart

## Etap 3. UI System

- wybrać finalne fonty
- ustalić system spacingu
- zaprojektować przyciski, badge, karty i like button
- ustalić theme states dla głównych sekcji

## Etap 4. Layout

- zbudować statyczną wersję strony bez zaawansowanych animacji
- dopracować mobile
- dopracować desktop

## Etap 5. Moduł „Co dziś pieczemy”

- zaprojektować kartę wypieku
- wdrożyć sezonowość
- wdrożyć UI polubień
- przygotować strukturę danych pod przyszłą administrację

## Etap 6. Motion

- wdrożyć hero reveal
- wdrożyć przejścia tła między sekcjami
- wdrożyć reveal tekstu i wypieków
- dodać mikrointerakcje kart i badge

## Etap 7. QA

- sprawdzić mobile, tablet i desktop
- sprawdzić polskie znaki i kodowanie UTF-8
- sprawdzić rytm scrolla i wydajność
- poprawić czytelność i spójność sekcji

## Kolejność robocza

1. Fundament
2. Content
3. UI System
4. Layout
5. Co dziś pieczemy
6. Motion
7. QA
