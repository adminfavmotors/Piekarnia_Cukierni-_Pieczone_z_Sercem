# Pieczone z Sercem Web

Frontend projektu `Pieczone z Sercem`, zbudowany na:

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Założenia

- język strony: polski
- priorytet: mobile first
- estetyka: ciepła, domowa, nowoczesna
- moduł `Co dziś pieczemy` przygotowany pod późniejszą rozbudowę

## Uruchomienie lokalne

```bash
npm install
npm run dev
```

## Ważne katalogi

- `src/app` — layout i strona główna
- `src/components` — komponenty współdzielone
- `src/data` — dane startowe strony

## Status

Obecna wersja zawiera:

- bazowy app shell
- globalny system kolorów i typografii
- pierwszą statyczną, mobile-first strukturę strony głównej
